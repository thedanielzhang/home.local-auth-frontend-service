# Frontend Template

A minimal React frontend template with TypeScript and Parcel bundler.

## Stack

- **React 19** - UI library
- **TypeScript** - Type-safe JavaScript
- **Parcel** - Zero-config build tool and dev server

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs the dev server at http://localhost:1234 with hot reloading.

## Production Build

```bash
npm run build
```

Outputs optimized static files to `dist/`.

## Configuration

Copy `.env.example` to `.env` and set:

- `API_URL` - Backend API URL (defaults to http://localhost:8000)
